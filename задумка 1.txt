Игра "РЭДИ ОР НОТ 2д"

Стиль: 2D сбоку, простой пиксельный (3-5 фреймов на анимацию) (обусловлено сложностью реализации)

Анимация: создание вручную или рендер спрайтов в blender?
    объекты под наклоном 15-30 градусов по горизонту в сторону камеры
    ноги отдельно анимируются (для сплита ходьбы и стрельбы)
    5 положений оружия  (0гр +-5гр +-10гр) (можно и больше между ними положений напихать)
    спрайты для рук общие по группам: основн.оруж, втор.оруж, спецовка

Звук????????????????????

Главное меню:
\ Играть
\ Настройки
\ Надоело играть

Настройки:
    \качество графики: картошка/нормальные/квантовый компутер
    \? прочие настройки для теста

    Качество графики:
        хз, мб просто изменение фреймрейта, если таковой будет,
        или кол-во лучей для просчёта зоны видимости игрока

Надоело играть: выход

Играть:
    \Шуточная ошибка: Данная игра находится в разработке. Доступные опции игры: потыкать главное меню \ОК: переход в нетыкабельное главное меню, после 2сек на него падает кирпич и появляется /игровое меню

    Игровое меню = карта с заданиями

    Карта с заданиями:
        тайл задания состоит из img и названия, все задания последовательно соедениненынененинеиенын (соединены) линией как-то так: VV (около 5 заданий, мб меньше, зависит от скорости разработки и наполнения контентом)

    После тыка на задание: затенение, \меню задания

    Меню задания:
        слева sidebar вкладки: миссия (вся её суть), карта, снаряжение, ГОТОВ

        Миссия:
            h1 Название
            p Справка о месте проведения операции
            p Предыстория (кто что захватил, или кто что курил, кто кого убил)

        Карта: схематичный набросок карты

        Снаряжение:
            topbar: боец, магазЫн

            Боец:
                1 основное оружие (mp5 ak47 m4a1 nova? saiga?)
                2 второе оружие (нет(+1 маг для осн), глок, дыгл)
                3 гранаты (флешка, слезоточивая)
                4 спец штука (локер(быстрая открывашка дверей), камера(для просмотра комнаты через щель снизу двери), щит?)
                5 броник (нет(+скорость), лёгкий, средний, тяжёлый)
                6 шлем (лёгкий, средний, тяжёлый(пубгерский), противогаз)
                7 разгруска (нет(1маг 1гр), малая(2маг 2гр), средняя(3маг 4гр), больш(6маг 6гр))

            МагазЫн: за деньги и опыт с миссий. опыт влияет на открытие возможности купить что-то
                topbar: осн.оруж, втор.оруж, гран, спец, броник, шлем, разгр
                
                условная шкала опыта:
                    10 50 100 200 300 400 500 1000
                    == == === === === === === ====
                    |  |  |   |   |   |   |   \разгруска(больш), nova?, saiga?
                    |  |  |   |   |   |   \щит?, броник.шлем(тяжёлый), ak47
                    |  |  |   |   |   \разгруска(средняя)
                    |  |  |   |   \слезоточивая, противогаз, камера, дыгл, m4a1
                    |  |  |   \локер, броник.шлем(средний)
                    |  |  \разгруска(малая), mp5
                    |  \броник.шлем(лёгкий)
                    \флешка
        
        ГОТОВ:
            старт \задания
            
            Задания:
                для начала 5 шт, мб меньше. если меньше, то контента будет меньше (условно до 200 опыта). в целом, сделать хотяб 1 задание с рабочей прогрессией будет успехом
                нейтрализация врага: +5оп + 5оп(нож) + 10оп(пистолет)*мощность + 20оп(автомат)*мощность
                упущение врага (он сбежит): -5оп
                спасение гражданского: +5оп - 5оп*(1-%здоровья)
                выполнение задания: +10оп
                смерть гражданского -10оп (от заработанных в миссии (минимум 0))
                убийство гражданского -20оп (от заработанных в миссии (минимум 0))
                множитель награды: 1 + 0.2(никто из гр не умер) + 0.2(враги без сознания (живы)) + 
                
                1: ресторанчик на окраине города
                    4 врага (2 с глок, 2 с нож), 6 гражданских
                2: частный дом в посёлке
                    2 врага (1 глок 1 ревик), 4 гражданских
                    требуют выкуп за заложников. 30сек промедления: -1гражданский
                пока что фантазии не хватает на выдумку заданий с плавным повышением сложности
